# cafe_lima_rasa.sql
Sistem Database Cafe Lima Rasa merupakan sistem basis data yang dirancang untuk membantu pengelolaan operasional bisnis cafe secara terstruktur dan terintegrasi. Sistem ini berfungsi untuk mengelola data menu makanan dan minuman, pelanggan, karyawan, pesanan, transaks pembayaran, stok bahan baku, serta hubungan dengan supplier.
Dengan menggunakan sistem database ini, proses pencatatam pesanan dan transaksi dapat dilakukan denga rapi dan efisien. Selain itu, sistem juga mendukung pengelolaan stok bahan baku secara otomatis melalui penggunaan trigger dan function, sehingga dapat mengurangi kesalahan pencatatan. Database ini dirancang menggunakan konsep normalisasi hingga bentuk normal lanjut (3Nf - 5NF) serta dilengkapi dengan constraint, stored procedure, function, dan trigger untuk menjaga konsistensi dan keakuratan data.
Sistem ini diharapkan dapat membantu pihak kami dalam meningkatkan efisiensi kerja, keakuratan data, serta memudahkan pengambilan keputusan terkait operasional bisnis sehari-hari.
